## Baseline

Post-[ADR-014](decisions.md) merge, `main` carries:

**GMRFs.jl (~M1–M6 landed)** — graph, precision, sampling, log-density,
constraints, factorization cache, nullspace, marginal variances via
`SelectedInversion.jl` ([ADR-012](decisions.md)), Makie recipes,
oracle fixture for `qinv_rw2`. Reference-impl still present as `:dense`
method.

**LatentGaussianModels.jl (~M1–M5 landed)** — components (Intercept,
IID, RW1, RW2, AR1, Besag, BYM2); likelihoods (Gaussian, Poisson,
Binomial); priors (PC, BYM2 φ); inference (EmpiricalBayes, Laplace with
hard constraints, INLA outer loop); integration schemes
(CCD/Grid/GaussHermite); posterior marginals + R-INLA-shaped
accessors; diagnostics (DIC/WAIC/CPO/PIT); Scotland BYM2 oracle;
simplified-Laplace skew correction (in tension with [ADR-006](decisions.md)).

**INLASPDE.jl (~M1–M6-A landed)** — FEM assembly (C, G₁, C̃, G₂),
mass lumping, precision for α∈{1,2} ([ADR-013](decisions.md)),
DelaunayTriangulation mesh generation, boundary handling, MeshProjector
(sparse barycentric), PC-Matérn prior, SPDE2 component, Meuse oracle
+ fmesher parity fixtures, GeoInterface + Makie extensions.

**INLASPDERasters.jl (landed)** — raster→mesh extraction (bilinear +
nearest), predict + quantile rasters, Makie extension.

**Scaffold-only** — `LGMFormula.jl`, `LGMTuring.jl`, `GMRFsPardiso.jl`.

- [x] ~~`Leroux` CAR component + oracle test (Sardinia or Ohio).~~
      **Landed 2026-04-24** (`e453eb2`). Proper CAR via convex
      combination Q = τ((1-ρ)I + ρR) with R the graph Laplacian; two
      hyperparameters (log τ, logit ρ). New `LogitBeta(a, b)` prior

- [x] ~~`BYM` (non-reparameterised form) — distinct from `BYM2`, R-INLA
      still ships both. Oracle test against `model = "bym"`.~~
      **Landed 2026-04-24** (`e453eb2`). Classical iid + besag
      convolution; latent layout [v; u] length 2n, blockdiag precision
      diag(τ_v·I, τ_u·Q_besag); per-component sum-to-zero on the
      spatial block only. Synthetic Poisson recovery on an 8×8 grid
      green. Oracle fixture against `model = "bym"` still owed as a
      B-tail.
- [x] ~~`Seasonal` LGM component wrapping `SeasonalGMRF`.~~
      **Landed 2026-04-24** (`d04da8d` GMRFs tier, `2e62641` LGM tier).
      SeasonalGMRF adds Q = τ·B'B penalising every period-s consecutive
      sum (Rue & Held 2005 §3.4.3); rankdef = s-1 with a constraint
      matrix whose row span matches null(Q) exactly. Thin LGM wrapper
      delegates to the GMRF. Laplace-enforced null-space projection
      verified to 1e-8; Gaussian synthetic recovery on period-6 × 36
      points green. Oracle fixture against `model = "seasonal"` still
      owed as a B-tail.
- [x] ~~`Generic0` and `Generic1` — delegation wrappers around
      `Generic0GMRF`, with `Generic1` applying the eigenvalue rescaling
      per [packages/GMRFs.jl/plans/plan.md M2 note](packages/GMRFs.jl/plans/plan.md).~~
      **Landed 2026-04-24** (`5f77cb7`). `Generic0` is a thin LGM
      wrapper (pass-through of rankdef + scale_model). `Generic1`
      rescales `R` at construction so `λ_max(R) = 1`. Tests verify
      agreement with direct `Generic0GMRF` calls and check
      `λ_max(c.R) ≈ 1` after Generic1 construction. The β-mixing
      flavour of R-INLA's `generic1` (convex combination of I and C̃
      with β ∈ (0,1)) is deferred beyond v0.1.
- [x] ~~Disconnected-graph Besag test (canonical dataset: Sardinia).~~
      **Landed 2026-04-24** (`9d110b0`). LGM-layer test covering (a)
      `constraint(Besag(g))` producing one sum-to-zero row per connected
      component, (b) `model_constraints` embedding those rows at the
      right column offsets in the stacked `x`, (c) `laplace_mode`
      enforcing per-component sum-to-zero on the MAP — the smoking-gun
      check against the silent Freni-Sterrantino failure mode — and
      (d) `inla(..., :grid)` producing finite summaries end-to-end.
      3-component hand-built graph (chain-4 / triangle-3 / chain-3);
      GMRFs.jl already carries its own disconnected-Besag coverage at
      the precision / sampling / constraint tier.
- [x] ~~`NegativeBinomial`, `Gamma` likelihoods with closed-form gradients.~~

## Risks not yet allocated

- **Selected-inversion edge cases at scale.** ADR-012 validated on
  band / random-SPD / Laplacian matrices. Real SPDE precisions on
  10⁴-vertex meshes have not been stress-tested. If
  `SelectedInversion.jl` has a memory or numerical surprise at that
  scale, Phase D slips by the effort to drop in a native Takahashi.
- **R-INLA version pinning.** `renv.lock` in `scripts/generate-fixtures/`
  pins R packages but not R itself. A user regenerating fixtures on a
  different R minor version may hit BLAS-driven numerical drift that
  fails tamper checks. Document explicitly or add a CI-only R version
  pin.
- **Matrix market for SPDE α=1.** v0.1 ships α=2 as the R-INLA default;
  α=1 is mathematically supported and tested but has no oracle fixture.
  Low priority but named so it's not forgotten.

## What this replan does not schedule

These are deferred beyond v0.1 and tracked only to prevent scope creep:

- Space-time Kronecker components.
- Joint likelihoods / misaligned multi-response models.
- Simplified Laplace full Rue-Martino correction (beyond the skew
  term already landed).
- Fractional-α SPDE (Bolin-Kirchner rational).
- GPU factorization via CUDSS.
- `inla.rgeneric` migration guide (the architecture makes it automatic;
  a guide is still owed).
